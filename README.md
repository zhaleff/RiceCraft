## Arch Linux
### Hyprland — Quickshell
Dotfiles where [Quickshell](https://quickshell.outfoxxed.me/) (a QML-based shell toolkit) is a primary or significant component. These are complete personal dotfile repositories, not standalone shells or frameworks.
- [notcandy001/Moonveil](https://github.com/notcandy001/Moonveil) — Hyprland dotfiles with a self-maintained Quickshell bar (Moonshell), barcode lockscreen, dynamic media controls, and a full QML control center. Requires Matugen.
- [caioax/lyne-dots](https://github.com/caioax/lyne-dots) — Full Arch rice managed with GNU Stow. Custom Quickshell bar, launcher, notifications, quick settings, and power menu in QML. 11-theme system with live switching and built-in wallpaper picker.
- [cxOrz/dotfiles-hyprland](https://github.com/cxOrz/dotfiles-hyprland) — ChromeOS-inspired Hyprland environment with Material Design 3 aesthetics. 87% QML — Quickshell handles the entire shell layer. One-command install script included.
- [0xfbad/dotfiles](https://github.com/0xfbad/dotfiles) — NixOS and Hyprland config using Quickshell for desktop overlays, Niri-style scrolling layouts, and animated wallpaper transitions.
- [anasgets111/dotfiles](https://github.com/anasgets111/dotfiles) — Comprehensive Hyprland/Niri dotfiles with a deeply customized Quickshell implementation (73% QML). Covers hardware monitoring, MPRIS controls, and a custom panel framework.
- [BelimFaux/qsdots](https://github.com/BelimFaux/qsdots) — Dotfiles dedicated strictly to Quickshell components. Acts as the complete desktop shell over Hyprland, built from scratch in QML. Mirrored from Codeberg.
- [MelonEncoder/dotfiles](https://github.com/MelonEncoder/dotfiles) — Cross-distro dotfiles for NixOS and Arch. Custom Quickshell QML scripts drive the primary shell components.
- [0Crazy-0/dotfiles](https://github.com/0Crazy-0/dotfiles) — Wayland and Hyprland rice that replaces Waybar entirely with a Quickshell-built system-level GUI layer.
- [shub39/dotfiles](https://github.com/shub39/dotfiles) — Clean and minimal Arch dotfiles with custom QML/Quickshell code for widgets and status indicators, no traditional panel daemon.
- [KingerNL/.dotfiles](https://github.com/KingerNL/.dotfiles) — Retro orange aesthetic Hyprland setup mixing Rofi with Quickshell elements.
- [caffeine-addictt/dotfiles](https://github.com/caffeine-addictt/dotfiles) — Personal config focused on QML-driven interfaces. Quickshell replaces standard desktop elements with reactive, custom-built panels.
- [EC2854/dotfiles](https://github.com/EC2854/dotfiles) — Modern rice experimenting with Quickshell for fluid animations and custom Wayland workspaces.
- [nvdson2005/quickshell-dots](https://github.com/nvdson2005/quickshell-dots) — Hyprland dotfiles with a series of evolving Quickshell styles listed from newest to oldest. Community-maintained install script.
- [bgibson72/yahr-quickshell](https://github.com/bgibson72/yahr-quickshell) — Complete Arch + Hyprland dotfiles with a Quickshell QML desktop featuring glassmorphism panels, 13 switchable themes synced across all apps, and a GPU-aware automated installer.
- [R7rainz/dotfiles](https://github.com/R7rainz/dotfiles) — Hyprland rice using Noctalia Shell — a minimal Wayland shell on Quickshell. IPC-controlled lockscreen, Zen Browser integration, PipeWire audio widgets.
- [Kanjurito/dotfiles](https://github.com/Kanjurito/dotfiles) — Personal Niri + Quickshell rice with Waybar, Eww, Fastfetch, and Alacritty. Lightweight and reproducible.

### Distro-Agnostic — Shell & Terminal
Universal shell and terminal configs that pair well with any WM or DE.

- [Gogh-Co/Gogh](https://github.com/Gogh-Co/Gogh) — 250+ terminal color schemes with one-line installers for Kitty, Alacritty, WezTerm, GNOME Terminal, and more. Used in rices of every kind.

- [catppuccin/catppuccin](https://github.com/catppuccin/catppuccin) — The dominant pastel color scheme of 2024–2026 ricing culture. Community-maintained ports for every app imaginable.
- [rose-pine/rose-pine-theme](https://github.com/rose-pine/rose-pine-theme) — Soft, warm pine and soho-inspired palette. Ports across terminals, editors, WM configs, and browsers. A staple in cozy, muted aesthetics.
- [zellij-org/zellij](https://github.com/zellij-org/zellij) — Modern terminal multiplexer with a clean default UI and extensive theming support. Increasingly used alongside Kitty and WezTerm in current rices.
