## Arch Linux

### Hyprland

- **[ESHAYAT102/dotfiles](https://github.com/ESHAYAT102/dotfiles)** — Minimal Omarchy rice running Catppuccin Mocha, fish shell, and Ghostty terminal. Clean and well put together. `by Eshayat Al-Wasiu`
- **[aloy555 — AM i A psYcHo?](https://working_on_it.com)** — Everything themed around a psycho aesthetic, including a custom-made wallpaper. `by aloy555`
- **[na-ive/nandoroid-shell](https://github.com/na-ive/nandoroid-shell)** — A Quickshell-based Hyprland shell inspired by Android custom ROMs, bringing a mobile UI language to the Linux desktop. `by na-ive`
- **[leaf — Hyprland Aura Soft](https://github.com)** — Hyprland setup built around the Aura aesthetic in a soft palette — gentle tones and a quietly elegant atmosphere. `by leaf`
- **[linuxmobile/hyprland-dots](https://github.com/linuxmobile/hyprland-dots)** — linuxmobile's first Hyprland rice, built around a Rosé Pine palette. A classic. `by linuxmobile`
- **[linuxmobile/hyprland-dots — old](https://github.com/linuxmobile/hyprland-dots)** — An older Hyprland iteration from linuxmobile, already showing the attention to detail that defined later setups. `by linuxmobile`
- **[zhaleff/BlackNode](https://github.com/zhaleff/BlackNode)** — A dark, minimalist Hyprland environment with charcoal backgrounds and sharp contrast. No bloat, just a polished daily driver. `by zhaleff`
- **[zhaleff/hyprcraft](https://github.com/zhaleff/hyprcraft)** — A meticulously crafted Hyprland setup for Arch, focused on aesthetics, consistency, and usability. Catppuccin Mocha colors throughout. `by zhaleff`
- **[caioax/lyne-dots](https://github.com/caioax/lyne-dots)** — Full Arch + Hyprland desktop with custom Quickshell components and the distinctive Lyne aesthetic. `by caioax`
- **[leaf — caelestiax](https://github.com/Axenide/Ambxst)** — Caelestiax frame with AI modules and extra integrations running on Hyprland. `by leaf`
- **[leaf — hyprland retro](https://github.com/Axenide/Ambxst)** — The Ambxst shell in a retro frame — a nostalgic take on the Hyprland desktop. `by leaf`
- **[anonymous — hyprland ambxst](https://github.com)** — Ambxst Quickshell on Hyprland with clean, well-designed modules. `by anonymous`
- **[Bixer234/Dots](https://github.com/Bixer234/Dots)** — A Hyprland setup built for fun and personal use. Simple, honest, and worth a look. `by Bixer234`
- **[vyrx-dev/symphony](https://github.com/vyrx-dev/symphony.git)** — A minimal, productivity-focused Arch + Hyprland setup built to be the environment vyrx-dev wished already existed. `by vyrx-dev`
- **[cebem1nt/dotfiles](https://github.com/cebem1nt/dotfiles)** — Modular and extensible Hyprland dots using Waybar, swaync, and rofi. Includes a Spicetify theme, wallpaper picker, and a light/dark theme switcher. `by cebem1nt`
- **[notcandy001/Moonveil](https://github.com/notcandy001/Moonveil)** — A calm Hyprland setup built for real workflow. Every detail tuned by hand — from keybindings to window rules — plus a barcode-style lockscreen. `by notcandy001`

### Sway

- **[vyrx-dev/dotfiles](https://github.com/vyrx-dev/dotfiles.git)** — A deliberately boring Sway setup focused on development work. This is where you actually get things done. `by vyrx-dev`

### Other

- **[AyushKr2003/niri-caelestia-shell](https://github.com/AyushKr2003/niri-caelestia-shell)** — The Caelestia shell ported to the Niri window manager, with a config editor, battery monitor, enhanced workspace bar, clipboard manager, and Niri IPC dashboard controls. `by AyushKr2003`
- **[linuxmobile — old rice with openbox](https://github.com/linuxmobile)** — An old Openbox rice from linuxmobile. `by linuxmobile`
- **[linuxmobile — openbox & android old rice](https://github.com/linuxmobile)** — One of the first rices linuxmobile shared in the unixporn community. A piece of history. `by linuxmobile`
